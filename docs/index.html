前言

这是一本关于生态统计学的教材，旨在为生态学研究者提供实用的数据分析方法。本书结合R语言，介绍生态学研究中常用的统计技术。

本书特点：

  • 面向生态学研究实际问题
  • 基于R语言实现
  • 包含大量生态数据案例
  • 循序渐进的教学安排

本书使用以下R包：

install.packages(c(
  "tidyverse", "vegan", "lme4", "ggplot2", 
  "bookdown", "knitr", "rmarkdown"
))
